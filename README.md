# Employee Attrition Analysis and Prediction

This project analyzes HR employee data to identify key factors 
influencing attrition and builds a Logistic Regression model 
to predict employee turnover risk.

## Tools Used
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Scikit-learn

## Key Steps
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis (Chi-square, t-test, correlation)
- Machine Learning Modeling
- Result Interpretation

## Dataset
IBM HR Employee Attrition Dataset

## Output
- Attrition prediction (Yes/No)
- Probability score for attrition risk
